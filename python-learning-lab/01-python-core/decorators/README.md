# Decorators

## Exercises

| File | Concepts | Difficulty |
|------|----------|------------|
| `01_basic_decorator.py` | `@functools.wraps`, timer, retry, memoize | ⭐⭐ |
| `02_class_decorators.py` | Decorators as classes, `__call__` | ⭐⭐⭐ |
| `03_real_world.py` | Rate limiter, circuit breaker, route registry | ⭐⭐⭐⭐ |

## Getting started

```bash
python 01_basic_decorator.py
```

You will see the output of the already-implemented examples.
Your job is to complete the functions marked with `TODO`.
